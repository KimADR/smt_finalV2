## Caractéristiques du PDF

✅ **Format professionnel** - Mise en page claire et structurée
✅ **Informations complètes** - Tous les éléments du calcul
✅ **Lisible** - Police Helvetica, tailles de police variables
✅ **Imprimable** - Dimensions standard A4
✅ **Formatage monétaire** - Nombres avec séparateurs (MGA)
✅ **Métadonnées** - Inclut la date et heure de génération
✅ **Nommage intelligent** - Nom du fichier basé sur l'entreprise

## Comment l'utiliser

1. Ouvrir le **Calculateur d'Impôt** (Tax Calculator)
2. Remplir les données financières:
   - Chiffre d'affaires annuel
   - Charges déductibles
   - Acomptes versés
3. Cliquer sur le bouton **"Télécharger (PDF)"**
4. Le PDF s'affiche dans le dossier de téléchargement
5. Peut être imprimé directement ou envoyé par email
